Ark Admin Manager

Ark Admin Manager, AAM for short, is a personal project of mine that aims to make the job of being a admin on a Ark server easier, it does this my replacing the most used rcon commands and command sequences with a visual gui.
this is a screenshot of the program, still in its alpha/beta phase. the program is downloadable on my github. the changelog for updates doesnt exist tho...
a little explainer of the gui:
in the top bar there are some general settings and the servers option this option is only for server owners tho, for more explanation on the account system see a bit further down.
the tab bar underneath that is where you navigate the program, most tabs are pretty self explanatory the imported items that show any mod items that users imported, the queue is a list of commands that the uses queued.

the account system

the account system is relatively simple, there are 2 types of accounts: owner and admin, an owner account can be created by just making a new account, an owner account sets up a server and can claim rcon ip addresses and ports, set commands that server admins cant use and of course create admin accounts, they do this by registering a admin my their steam id, an admin can then, if they have a steam id registered, create an account liked to that steam id.

this one is my default pair data type, I like using my own class instead of one from a lib so it is the same in every project, note to self: make a personal lib; 

/**
 * this class is the object for a data pair, the values of T and U dont have to be the same
 * @author Blackdragon2447
 *
 * @param <T> the first type in the pair
 * @param <U> the second type in the pair
 */
public class Pair<T, U>{

	T p1;
	U p2;

	/**
	 * makes a new pair
	 * @param a the value to be set for the first field
	 * @param b the value to be set for the second field
	 */
	public Pair(T a, U b){
		this.p1 = a;
		this.p2 = b;
	}

	/**
	 * same as the constructor but for an exsitent pair.
	 * @param a the value to be set for the first field
	 * @param b the value to be set for the second field
	 */
	public void setValue(T a, U b){
		this.p1 = a;
		this.p2 = b;
	}

	/**
	 * @return returns the pair.
	 */
	public Pair<T, U> getValue(){
		return this;
	}

	/**
	 * @return returns the first value of the pair
	 */
	public T getFirstValue() {
		return p1;
	}

	/**
	 * @return returns the second value of the pair
	 */
	public U getSecondValue() {
		return p2;
	}

	/**
	 * tries to make a csv vlaue of the pair
	 * @return returns the result of the pairs csv value
	 */
	public String GetcsvValue() {
		return p1.toString() + ", " + p2.toString();
	}

	@SuppressWarnings("unchecked")
	@Override
	public boolean equals(Object obj) {
		Pair<T, U> compObj = (Pair<T, U>) obj;
		return ((this.p1 == compObj.p1) && (this.p2 == compObj.p2));
	}

}

public static ArrayList<Pair<String, String>> readCSV(String path) throws IOException {

	BufferedReader br = new BufferedReader(new FileReader(path));

		String[] values = Line.split(",");

    function loadVideo(iframe) {
        $.getJSON(reqURL + iframe.getAttribute('cid'),
            function(data) {
                var videoNumber = (iframe.getAttribute('vnum') ? Number(iframe.getAttribute('vnum')) : 0);
                console.log(videoNumber);
                var link = data.items[videoNumber].link;
                id = link.substr(link.indexOf("=") + 1);
                iframe.setAttribute("src", "https://youtube.com/embed/" + id + "?controls=0&autoplay=1");
            }
        );
    }
    var iframes = document.getElementsByClassName('latestVideoEmbed');
    for (var i = 0, len = iframes.length; i < len; i++) {
        loadVideo(iframes[i]);
    }
